## 🚀 هەنگاوەکانی دامەزراندن

### هەنگاوی ١ — پێداویستییەکان
لە کۆمپیوتەرەکەت ئەمانە دابمەزرێنە:
- [Node.js](https://nodejs.org) وەشانی 18 یان نوێتر
- [Git](https://git-scm.com)

### هەنگاوی ٢ — Supabase (داتابەیس و ئەکاونت)
1. بڕۆ بۆ **supabase.com** و ئەکاونتێکی خۆڕایی دروست بکە
2. **New Project** دابگرە — ناوێک و وشەی نهێنیێکی داتابەیس دابنێ (Region: هەرچی نزیکترە، بۆ نموونە Frankfurt)
3. کە پڕۆژەکە ئامادە بوو، لە لای چەپ بڕۆ بۆ **SQL Editor**
4. ناوەڕۆکی فایلی `supabase/schema.sql` کۆپی بکە و لەوێ **Run** ی بکە ✅
5. بڕۆ بۆ **Project Settings → API** و ئەم دوو بەهایە هەڵبگرە:
   - `Project URL`
   - `anon public key`
6. (بۆ تاقیکردنەوەی خێرا) لە **Authentication → Providers → Email** دا، `Confirm email` بکوژێنەوە بۆ ئەوەی خەڵک بەبێ پشتڕاستکردنەوەی ئیمەیڵ بچنەژوورەوە. دواتر بۆ بەرهەمهێنان دەتوانیت بیکەیتەوە.

> 📱 **ژمارەی مۆبایل؟** Supabase پشتگیری چوونەژوورەوە بە SMS دەکات بەڵام پێویستی بە هەژماری Twilio یان MessageBird ە (پارەی تێدەچێت). بۆیە لە وەشانی یەکەمدا ئیمەیڵ بەکاردەهێنین — دواتر دەتوانین SMS زیاد بکەین.

### هەنگاوی ٣ — LiveKit Cloud (دەنگ)
1. بڕۆ بۆ **cloud.livekit.io** و ئەکاونتێکی خۆڕایی دروست بکە
2. پڕۆژەیەکی نوێ دروست بکە (بۆ نموونە: kurd-voice)
3. لە **Settings → Keys** دا ئەمانە هەڵبگرە:
   - `WebSocket URL` (بە `wss://` دەست پێدەکات)
   - `API Key` و `API Secret`

### هەنگاوی ٤ — ڕانکردن لەسەر کۆمپیوتەرەکەت
```bash
# لەناو فۆڵدەری پڕۆژەکەدا:
cp .env.example .env.local
# فایلی .env.local بکەوە و بەهاکانی Supabase و LiveKit تێیدا دابنێ

npm install
npm run dev
```
ئینجا لە وێبگەڕەکەتدا بکەوە: **http://localhost:3000** 🎉

بۆ تاقیکردنەوەی دەنگ: دوو تابی جیاواز (یان دوو ئامێر) بکەوە، دوو ئەکاونتی جیا دروست بکە، هەردووکیان بچنە هەمان ڕووم — دەنگی یەکتری دەبیستن!

### هەنگاوی ٥ — بڵاوکردنەوە لەسەر ئینتەرنێت (Vercel)
1. پڕۆژەکە بخەرە سەر **GitHub**:
```bash
git init
git add .
git commit -m "Kurd Voice v1"
# لە github.com دا repo یەکی نوێ دروست بکە، ئینجا:
git remote add origin https://github.com/USERNAME/kurd-voice.git
git push -u origin main
```
2. بڕۆ بۆ **vercel.com** → **Add New Project** → repo کەکەت هەڵبژێرە
3. لە بەشی **Environment Variables** دا هەر پێنج بەهاکەی ناو `.env.local` زیاد بکە
4. **Deploy** دابگرە — دوای خولەکێک ئەپەکەت لەسەر لینکێکی گشتی بەردەستە! 🌍

> 🔒 **گرنگ:** مایک لە وێبگەڕدا تەنها لەسەر HTTPS کاردەکات — Vercel خۆکارانە HTTPS ت پێدەدات، کێشە نییە.

## 🎙️ چۆن کاردەکات؟

- **ڕووم دروست دەکەیت** → خۆکارانە دەبیتە ئەدمین و مایکت کراوەیە 👑
- **کەسێک جۆین دەبێت** → مایکی داخراوە، دەچێتە بەشی گوێگرەکان
- **✋ دەست بەرز دەکاتەوە** → لای ئەدمین دوگمەی "ڕێگەپێدان" بە زێڕینی دەردەکەوێت
- **ئەدمین ڕێگە دەدات** → سێرڤەر تۆکنێکی نوێ بە مۆڵەتی قسەکردن دەداتێ و دەچێتە سەر ستەیج
- مۆڵەتی مایک **لە سێرڤەرەوە** پشکنین دەکرێت (فایلی `app/api/livekit-token/route.js`) — کەس ناتوانێت فێڵ بکات

## 📁 پێکهاتەی پڕۆژەکە
```
kurd-voice-app/
├── app/
│   ├── page.js                    ← لاپەڕەی سەرەکی (تابەکان، ڕوومەکان، پرۆفایل)
│   ├── layout.js                  ← چوارچێوەی گشتی (RTL + فۆنتەکان)
│   ├── globals.css                ← ڕەنگ و ستایلەکان
│   └── api/livekit-token/route.js ← دروستکردنی تۆکنی دەنگ + پشکنینی مۆڵەت
├── components/
│   ├── ui.js                      ← خۆری ڕۆژ، ئەڤاتار، دوگمە...
│   ├── AuthScreen.js              ← چوونەژوورەوە / خۆتۆمارکردن
│   ├── RoomView.js                ← ژووری دەنگ (ستەیج + گوێگرەکان + چات)
│   └── FriendsTab.js              ← هاوڕێکان + چاتی تایبەت + پەیوەندی
├── lib/supabase.js                ← پەیوەستبوون بە Supabase
├── supabase/schema.sql            ← سکیمای داتابەیس (جارێک ڕانی دەکەیت)
└── .env.example                   ← نموونەی کلیلە نهێنییەکان
```
